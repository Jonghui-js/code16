# </> code16.net 개발 요약

## 웹사이트

[www.code16.net](http://www.code16.net)

## 기술

- React / Redux
- node.js / express
- mongoDB - NoSQL
- REST API
- AWS ec2 Linux / Route 53

## 기능

- 회원가입, 로그인, 회원탈퇴
- 글(댓글) 작성, 수정, 삭제
- 마이페이지 내 글, 댓글 확인
- 유형별로 글 모아보기
- 메인 페이지 유형별 static 확인
